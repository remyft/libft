# Libft
_Multiple basic C and other usefull functions_

## Build & Run

```bash
git clone https://github.com/remyft/libft
cd libft && make
```
This will create a libft.a file that can be used in any of your .c files.

Example:
```bash
gcc -o test_libft main.c libft.a
./test_libft
```

## New method learned

* String manipulation
* Simple linked list
* Memory allocation
