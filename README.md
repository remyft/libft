# Libft
_Multiple basic C and other usefull functions_

## Build & Run

```bash
git clone https://github.com/remyft/libft
cd libft && make
```
This will create a libft.a file that can be used in any of your .c files (cf. main.c).

## New method learned

* String manipulation
* Simple linked list
* Memory allocation
